# Toyota Supra Video Assets

This folder should contain video files for each car color variant.
The website will display these videos in the hero section, changing based on the selected color.

## Required Video Files

Add the following MP4 video files to this folder:

1. `supra_red.mp4` - Video of red Toyota Supra
2. `supra_blue.mp4` - Video of blue Toyota Supra
3. `supra_white.mp4` - Video of white Toyota Supra
4. `supra_black.mp4` - Video of black Toyota Supra
5. `supra_yellow.mp4` - Video of yellow Toyota Supra

## Video Specifications

For best results, use:
- **Format:** MP4 (H.264 codec)
- **Resolution:** 1920x1080 or 1280x720
- **Duration:** 5-15 seconds (for looping)
- **File Size:** Keep under 10MB per video for fast loading

## Notes

- If a video file is missing, the website will gracefully fall back to displaying the static image
- Videos auto-play on mute and loop continuously
- Videos are responsive and will scale to fit the container
